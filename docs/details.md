## フレームワーク

**Q: SolidBaseはドキュメントサイト向けフレームワーク（サイドバー・テーマ切替・MDX機能など）です。詩表示サイトとして作り直す際、どちらを選びますか？**

**A: SolidStart素（SolidBase削除）**

- `@kobalte/solidbase` を dependencies から削除
- `@iconify-json/ri`、`@solidjs/meta` も合わせて削除
- `app.config.ts` から `withSolidBase()` を削除
- `app/src/app.tsx` の `SolidBaseRoot` を削除
- `app/src/entry-server.tsx` の `getHtmlProps()` を削除
- `.mdx` ファイルはすべて `.tsx` に置き換え

## ナビゲーションUX

**Q: 「次へ」操作のUXはどれにしますか？（ボタン / 画面全体クリック / スワイプ）**

**A: 画面全体クリック**

- 画面全体をクリック/タップすることで次の詩へ進む
- カーソルを `pointer` にして「クリックできる」ことを示す
- スマートフォンでのタップ操作にも対応

---

## デザイン

**Q: デザインの方向性を選んでください。**

**A: ミニマル・白**

- 背景色: `#ffffff`（白）
- テキスト色: `#1a1a1a`（ほぼ黒）
- フォント: Noto Serif JP（serif系、Google Fonts）
- 余白: 大きめ
- 装飾: 最小限
- 縦書き: なし（横書き）
