# Rester への詩・短歌の追加方法

## はじめに

Rester は没後70年以上経過した故人の詩・短歌を表示するサイトです。
コンテンツはすべてパブリックドメインの作品に限定されています。

---

## 著作権について

### 掲載条件

- 著作者が**没後70年以上**経過していること（日本の著作権法に準拠）
- 2026年現在、**1955年以前に没した著者**の作品が対象

### 確認方法

1. 著者の没年を Wikipedia または国立国会図書館で確認
2. `(没年) + 70 ≤ 現在の年` であることを確認
3. 例: 1942年没 → 2012年にパブリックドメイン入り → 2026年現在 ✅

## 詩の追加手順

### 1. リポジトリをフォーク・クローン

```bash
git clone git@github.com:YOUR_USERNAME/rester.git
cd rester
git checkout -b add/poem-AUTHOR_NAME
```

### 2. `app/src/data/poems.json` を編集

配列の末尾に新しい作品を追加します。

```json
{
  "id": "021",
  "body": "詩の本文をここに記入\n改行は \\n で表現",
  "author": "著者名",
  "year": 1920,
  "genre": "短歌",
  "source": "出典の作品集名"
}
```

### フィールド仕様

| フィールド | 必須 | 説明 |
|----------|------|------|
| `id` | ✅ | 3桁のゼロ埋め数字。既存の最大値+1を使用（例: "021"） |
| `body` | ✅ | 詩・短歌の本文。改行は `\n` で表現 |
| `author` | ✅ | 著者名（漢字表記） |
| `year` | - | 作品の制作・発表年。不明な場合は `null` |
| `genre` | - | `"俳句"`, `"短歌"`, `"詩"` のいずれか |
| `source` | - | 出典（作品集名・雑誌名など） |

### 本文の書き方

**俳句（5-7-5）**: 句切れを全角スペース（　）で区切る

```
"古池や　蛙飛びこむ　水の音"
```

**短歌（5-7-5-7-7）**: 上の句と下の句を `\n` で改行

```
"はたらけど　はたらけど猶　わが生活\n楽にならざり　ぢつと手を見る"
```

**詩（自由形式）**: 行ごとに `\n` で改行

```
"ふらんすへ行きたしと思へども\nふらんすはあまりに遠し"
```

### 3. バリデーションの実行

```bash
node scripts/validate-poems.mjs
```

### 4. Pull Request の作成

- PRタイトル例: `feat: 石川啄木「千鳥」を追加`
- PRテンプレートのチェックリストをすべて完了してから提出

---

## 注意事項

- 現代語訳や改変版は掲載しないでください
- 旧仮名遣い・旧字体はそのまま保存してください
- 一度のPRで追加できる作品数に制限はありませんが、同一著者でまとめることを推奨します
